# Instrutions

## User

### Login

```
user login <username>
```
### View current user

```
user view
```

## Item

### Add Item

```cmd
item create <name> <price> <selling_price>
```

### Get all item

```cmd
item all
```

### View a item

```cmd
item view <id>
```

## Order

### Place order

```cmd
order place <id>
```
